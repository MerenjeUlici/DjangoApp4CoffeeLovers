# DjangoApp4CoffeeLovers
This app will be a locator for coffee in Skopje

Python version: 3.8.14
Django version: 3.2

To get started:

```
docker-compose up
```


## Useful Django commands:
Run migrations:
```
python manage.py makemigrations
python manage.py migrate
```

Create super user:
```
python manage.py createsuperuser
```
